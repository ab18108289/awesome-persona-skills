## 收录范围 / 不收录范围

**收录：**
- 明显具备人物感、角色感、人格视角或神秘人格体验的 GitHub 公开项目。
- 标准 `Skill / Agent Skill` 项目。
- 少量高质量人格化 AI repo，作为目录补充。

**不收录：**
- 纯工具链、纯 workflow、纯工程脚手架。
- 没有人设叙事、只有功能包装的普通 AI 仓库。
- 与本库主轴无关的泛资源集合。
